# Career Portfolio Details

## Overview
This is my personal career portfolio website, showcasing my skills, projects, experience, and professional background. The website provides an interactive and engaging way to learn more about me and my work.

# Features
- About Me: A section introducing who I am and my career journey.
- Projects: A showcase of my past and current projects with descriptions.
- Skills: A list of technologies and tools I specialize in.
- Contact Form: Allows visitors to reach out to me directly.
- Responsive Design: Optimized for both desktop and mobile devices.

## Technologies Used
- Front-End: HTML, CSS, JavaScript  
- Back-End: PHP  
- Database: MySQL  
- Server: XAMPP (Apache & MySQL)
